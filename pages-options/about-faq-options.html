About — FAQ Section Options

Four layout approaches for 12 FAQ items. All are functional accordions — click any question to see the answer expand. The first open item in each option shows a sample answer to demonstrate the expanded state.

Option A
Single column — clean line accordion
The industry standard. One column, full width (780px max), simple horizontal lines between items. No cards, no boxes — just clean typography and borders. The "+" sits in a tinted circle for easier click targeting. Most accessible, most scannable, answers expand cleanly without layout shifting. Works perfectly for 12 items.

Option B
Single column — grouped by category
Same single-column accordion but questions organized into 3 category groups with colored labels: "Method & approach" (coral), "Training path & certification" (pink), "Practical details" (purple). Items in white cards with subtle border. Visitors can scan category headers to find their question type immediately. The colored labels tie back to the brand's 3-color system.

Option C
Split layout — sticky heading left, questions right
Two-column editorial split: the heading, intro text, and a "Still have questions?" CTA link sit in a sticky left column. The accordion runs in the right column. As the visitor scrolls through questions, the heading stays visible. Uses the full container width efficiently. The left column provides context without competing with the questions. My recommendation — the most distinctive layout here.

Option D
Tabbed panel — filter by category
All questions inside a single white card. Three tabs across the top let visitors filter by topic: "Method & approach," "Training & certification," "Practical details." Only one category visible at a time — reduces overwhelm from 12 items to 3-5 per view. The tabs use the brand pink for the active state. Cleanest option for a large FAQ set.
