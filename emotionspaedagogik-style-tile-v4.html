19 · Design Principles
Do & don't — quick reference

✓ Do

  • Use large, overlapping SVG blobs that bleed off-screen
  • Make headlines emotionally dominant — clear size contrast
  • Use gradients in color bands, not flat colors
  • Add semi-transparent blob overlays inside color bands
  • Keep body text small and muted (0.86-0.95rem, #6B6B6B)
  • Use generous whitespace — sections breathe at 100px+
  • Feel editorial / magazine, not SaaS dashboard
  • Include hover states on all interactive elements
  • Use cream (#F7F3EE) as page background
  • Cards: 8px radius, 32px padding, near-invisible shadow

✗ Don't

  • Use border-radius: 50% circles as "blobs"
  • ALL CAPS for headlines
  • Stack two colored sections back-to-back
  • Use pure white (#FFF) as page background
  • Make body text compete with headlines
  • Use prominent drop shadows on cards (12px+)
  • Keep blobs small, contained, or separated
  • Default to centered text (left-align is default)
  • Use tight spacing — this is not a compact UI
  • Neutralize the brand's warmth "for cleanliness"
